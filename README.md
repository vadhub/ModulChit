# ModulChit
## study of algorithms
This application is needed for the study of algorithms through visualization.
Algorithms that are already in Google Play:
- GCDE
- Fast exponentiation
- Multiplicative group
- RSA endcrypt and decrypt
- Binary search
- Bubble sort
- Insertion sort
- Shell sort
- Qui—Åk sort
